# Gulp + Sass + Bootstrap Boilerplate

## Gulp plugins

-  gulp-htmlmin
-  gulp-uglify
-  gulp-rename
-  gulp-concat
-  gulp-imagemin
-  gulp-image-data-uri
-  gulp-sass
-  gulp-minify-css
-  gulp-purifycss
-  gulp-watch
-  browser-sync

## Gulp Tasks

1.  `gulp css`: compile sass to css + minify
2.  `gulp js`: concat + minify scripts
3.  `gulp img`: compress images
4.  `gulp html`: minify html
5.  `gulp sync`: browsersync + watch
